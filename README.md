# LifeLine – Dossier Médical d’Urgence

LifeLine est une application web destinée à faciliter l’accès rapide aux informations médicales essentielles d’une personne en cas d’urgence, à l’aide d’un QR Code médical.

Ce projet est réalisé dans le cadre du module PROJET PROFESSIONNALISANT – Applications Web Progressives (PWA)​

# Objectif du projet

L’objectif de LifeLine est de permettre aux secouristes ou aux personnes présentes d’accéder rapidement à des informations médicales importantes lorsque la personne concernée ne peut pas communiquer (accident, malaise, perte de connaissance).



# Principe de fonctionnement

1. L’utilisateur crée son profil médical via l’application.
2. Un QR Code personnel est généré.
3. Ce QR Code est porté sur un bracelet ou une carte médicale.
4. En cas d’urgence, le QR Code est scanné.
5. Les informations médicales nécessaires sont affichées pour faciliter une prise en charge rapide.
